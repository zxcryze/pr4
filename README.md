Система онлайн-записи - реализация слоя абстракции доступа к базе данных с использованием паттерна Repository и PDO.

## 📋 Описание проекта

Реализован уровень доступа к данным для системы онлайн-записи. Включает:

- Подключение к БД через PDO (Singleton)
- Базовый абстрактный репозиторий с общими методами
- Конкретные репозитории для таблиц: `clients`, `services`, `specialists`
- Кастомное исключение `RepositoryException`
- Подготовленные выражения для защиты от SQL-инъекций
- Транзакции для операций с несколькими таблицами
- Демонстрационный скрипт `demo.php`
